Privacy Notice

For statistical purposes only, Southwestern collects the following data:

  • IP addresses of users who visit our Web site
  • Dates of average Web site activity
  • Most viewed web pages on Southwestern’s site

We use these statistics only for improvement purposes for Southwestern’s Web site, not to identify any individual users. We do NOT enable “cookies” on Southwestern’s website. Personal information submitted by a user via phone, fax, or e-mail is not distributed to any parties outside of Southwestern.

We do provide links to other Government Web sites, to assist the user in obtaining official Government information. However, once the user has left Southwestern’s Web site, the user is then subject to the privacy and security policy of that site.
